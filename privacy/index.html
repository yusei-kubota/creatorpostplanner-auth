プライバシーポリシー

CreatorPostPlanner | 最終更新日: 2026年6月28日

CreatorPostPlanner（以下「本アプリ」といいます）は、SNSクリエイターがYouTubeおよびInstagramの 投稿予定、目標、投稿結果を管理するためのiOSアプリです。本ポリシーは、本アプリにおける情報の取得、 利用、保存、第三者提供、削除方法について説明するものです。

1. はじめに

本アプリは、ユーザーのプライバシーを尊重し、SNS連携に必要な範囲でのみ情報を取得します。 YouTube、Google、Instagram、Metaなどの外部サービスには、それぞれのプライバシーポリシーと利用規約が適用されます。

2. 取得する情報

2.1 ユーザーが入力する情報

  • 目標数
  • 投稿予定
  • タイトル、URL、メモ
  • 手動で追加したサムネイル画像

2.2 SNS連携により取得する情報

区分 取得する情報 取得方法
YouTube連携 チャンネル名、チャンネルハンドル、チャンネル登録者数、投稿動画のタイトル、投稿日、URL、サムネイル、再生数、いいね数、コメント数 ユーザーがGoogle OAuthで許可した後、YouTube Data API v3から読み取り専用で取得
Instagram連携 ユーザーネーム、フォロワー数、投稿のキャプション、投稿日、URL、サムネイル、再生数、いいね数、コメント数 ユーザーがInstagram OAuthで許可した後、Instagram APIから取得
ユーザー入力情報 目標数、投稿予定、タイトル、URL、メモ、手動で追加したサムネイル画像 ユーザーによるアプリ内入力
ログイン維持情報 OAuthの更新に必要なトークン 各SNSのOAuth認証後に取得し、iOS Keychainに保存

3. 利用目的

本アプリは、取得した情報を以下の目的で利用します。

  • ホーム画面でSNSごとの現在数、目標数、進捗を表示するため
  • カレンダーに投稿予定および投稿済みコンテンツを表示するため
  • 投稿ごとのタイトル、投稿日、URL、サムネイル、再生数、いいね数、コメント数を確認できるようにするため
  • ユーザーが入力したURL、メモ、目標を保存・編集できるようにするため
  • ユーザー本人の投稿結果を分析し、投稿管理を補助するため
  • 不具合対応、問い合わせ対応、セキュリティ維持のため

4. 第三者サービスとの通信

本アプリは、以下の目的で外部サービスと通信します。

サービス 目的 送信・取得されるデータ
YouTube Data API v3 YouTubeチャンネルと投稿動画の読み取り チャンネル情報、投稿動画情報、再生数、いいね数、コメント数、サムネイル
Instagram API Instagramプロアカウントと投稿情報の読み取り ユーザーネーム、フォロワー数、投稿情報、再生数、いいね数、コメント数、サムネイル
Logo.dev SNSやサービスロゴの表示 ロゴ取得に必要なサービスのドメイン名

4.1 YouTube API Servicesの利用

本アプリは、YouTube Data API Servicesを使用します。YouTube連携では https://www.googleapis.com/auth/youtube.readonly の読み取り専用スコープを使用し、 ユーザー本人のYouTubeチャンネルおよび投稿動画の情報を取得します。取得したYouTubeデータは、 本アプリ内でチャンネル登録者数、投稿カレンダー、投稿別の再生数・いいね数・コメント数、 サムネイル、分析結果を表示するためだけに使用します。

本アプリは、YouTube上の動画の投稿、編集、削除、コメント投稿、チャンネル設定の変更を行いません。 また、取得したYouTubeデータを第三者へ販売せず、第三者広告配信や信用評価のために使用しません。 Google APIから受け取った情報の利用は、 Google API Services User Data Policy （Limited Use要件を含みます）に従います。

YouTube連携機能を利用する場合、ユーザーには YouTube利用規約 も適用されます。 ユーザーは、Googleアカウントの アクセス権限ページ から、いつでも本アプリのGoogleアカウントへのアクセスを取り消すことができます。 Googleのプライバシーポリシーは https://policies.google.com/privacy をご確認ください。

4.2 Instagram APIの利用

本アプリは、Instagram APIを使用し、ユーザーが許可したInstagramプロアカウントのプロフィール情報、 投稿情報、投稿に関する基本的な指標を取得します。取得した情報は、投稿管理、カレンダー表示、 分析のために使用します。

本アプリは、現行バージョンではInstagramのDM機能を提供せず、DM内容の取得、保存、送信、分析を行いません。

5. データの保存と削除

現行バージョンでは、取得したSNSデータ、投稿予定、目標、メモは主にユーザーの端末内に保存されます。 OAuthのログイン維持に必要なトークンはiOS Keychainに保存されます。

本アプリは、ユーザーがSNS連携を解除する、アプリ内データを削除する、またはアプリを削除するまで、 アプリの機能提供に必要な範囲で情報を保持します。

ユーザーは、アプリ内のログアウトまたはSNS削除操作により、端末内に保存された該当SNSのログイン情報と関連データを削除できます。 Googleアカウント側からアクセス権を取り消すこともできます。詳しくは データ削除案内をご確認ください。

6. 第三者提供・販売・広告利用

本アプリは、取得したYouTubeデータ、Instagramデータ、ユーザー入力情報を第三者へ販売しません。 また、現行バージョンでは、取得したSNSデータを第三者広告配信、信用評価、またはAIモデル学習のために提供しません。

法令に基づく場合、ユーザーの同意がある場合、または不正利用・セキュリティ対応に必要な場合を除き、 ユーザーのSNSデータを第三者に開示しません。

7. 安全管理措置

本アプリは、ログイン維持に必要なトークンをiOS Keychainに保存し、必要な範囲でのみAPIにアクセスします。 また、YouTube連携では読み取り専用スコープを使用し、不要な権限を要求しないよう努めます。

8. 未成年者の利用

本アプリは、13歳未満の児童を対象としていません。未成年の方が利用する場合は、 保護者の同意を得たうえで利用してください。

9. ポリシーの変更

本ポリシーは、法令、外部サービスの仕様変更、アプリ機能の変更に応じて改定される場合があります。 重要な変更がある場合は、本ページまたはアプリ内で告知します。

10. 商標および第三者ブランドについて

本アプリ内で表示されるYouTube、Instagram、その他サービス名、ロゴ、アイコン、商標は、 それぞれの権利者に帰属します。本アプリは、Google、YouTube、Meta、Instagram、Logo.devと 提携、後援、承認されたものではありません。

これらの名称やロゴは、ユーザーが連携先SNSやアカウントを識別しやすくする目的で使用します。 ロゴ画像は Logo.dev を通じて取得する場合があります。
