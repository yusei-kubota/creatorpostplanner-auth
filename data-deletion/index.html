データ削除案内

最終更新日: 2026年6月28日

このページでは、CreatorPostPlannerで保存されたSNS連携情報、投稿管理データ、 OAuthアクセス権を削除または解除する方法を説明します。

1. アプリ内でSNS連携データを削除する方法

アプリ内から、連携済みSNSごとのデータを削除できます。

  1. CreatorPostPlannerを開きます。
  2. ホーム画面で削除したいSNSカードを開きます。
  3. 詳細画面の「ログアウト」または「削除」を選択します。
  4. 確認画面で削除を実行します。

この操作により、該当SNSのログイン維持情報、保存済みの目標、URL、メモ、 取得済み投稿データが端末から削除されます。

2. アプリ全体のデータを削除する方法

iOSのホーム画面からCreatorPostPlannerを削除すると、アプリ内に保存された投稿予定、 目標、メモ、取得済みデータが端末から削除されます。

端末からアプリを削除しても、GoogleやInstagram側で発行済みのアクセス権が残る場合があります。 完全に連携を解除したい場合は、以下の外部サービス側の解除手順も実行してください。

3. Google / YouTubeのアクセス権を取り消す方法

YouTube連携を完全に解除するには、Googleアカウント側でCreatorPostPlannerのアクセス権を取り消してください。

  1. Googleアカウントのアクセス権限ページを開きます。
  2. CreatorPostPlannerを選択します。
  3. 「アクセス権を削除」または同等の解除操作を実行します。

この操作により、CreatorPostPlannerは以後、そのGoogleアカウントのYouTube情報を取得できなくなります。

アプリ内でYouTube連携を削除した場合、端末内に保存されたYouTubeのチャンネル情報、投稿情報、 サムネイルURL、再生数、いいね数、コメント数、OAuthトークンは削除されます。

4. Instagram / Metaの連携を解除する方法

Instagram連携を完全に解除するには、InstagramまたはMeta側のアプリ連携管理画面から CreatorPostPlannerのアクセス権を削除してください。

  1. InstagramまたはMetaの設定画面を開きます。
  2. アプリとウェブサイト、またはビジネス統合に関する設定を開きます。
  3. CreatorPostPlannerを選択します。
  4. アクセス削除または連携解除を実行します。

この操作により、CreatorPostPlannerは以後、そのInstagramアカウントの情報を取得できなくなります。

5. メールで削除を依頼する方法

アプリ内操作ができない場合、または削除について確認したい場合は、以下のメールアドレスまでご連絡ください。

creatorpostplanner@gmail.com

お問い合わせ時には、可能な範囲で以下を記載してください。

  • 削除対象のSNS種別（YouTubeまたはInstagram）
  • アプリ内に表示されているアカウント名またはユーザーネーム
  • 削除したいデータの種類（SNS連携、投稿予定、メモ、全データなど）

受領後、本人確認や削除対象の確認が必要な場合があります。確認後、合理的な範囲で速やかに対応します。 YouTube API Servicesから取得したユーザーデータについては、確認完了後、原則として7日以内の対応を目安とします。

6. 削除後に残る可能性がある情報

外部SNS自体に保存されている投稿、チャンネル、プロフィール、コメント、いいねなどは、 CreatorPostPlannerから削除しても各SNS上には残ります。外部SNS上のデータ削除は、 YouTube、Google、Instagram、Metaの各サービス上で行ってください。
