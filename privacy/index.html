プライバシーポリシー

最終更新日: 2026年6月28日

CreatorPostPlannerは、SNSクリエイターが投稿予定、目標、投稿結果を管理するためのiOSアプリです。 本ポリシーでは、CreatorPostPlannerにおけるデータの取得、利用、保存、削除について説明します。

取得する情報

ユーザーが連携した場合、以下の情報を取得します。

  • YouTube: チャンネル名、ハンドル、チャンネル登録者数、投稿動画のタイトル、投稿日、URL、サムネイル、再生数、いいね数、コメント数
  • Instagram: ユーザーネーム、フォロワー数、投稿のキャプション、投稿日、URL、サムネイル、再生数、いいね数、コメント数
  • ユーザー入力情報: 目標数、投稿予定、タイトル、URL、メモ、サムネイル画像

利用目的

  • ホーム画面でSNSごとの現在数と目標数を表示するため
  • カレンダーに投稿予定と投稿済みコンテンツを表示するため
  • 投稿ごとの再生数、いいね数、コメント数を分析するため
  • ユーザーが入力したURLやメモを管理するため

保存方法

現在のバージョンでは、取得した投稿情報、目標、メモはユーザーの端末内に保存されます。 OAuthのログイン維持に必要なトークンはiOSのKeychainに保存されます。 本アプリは、取得したYouTubeまたはInstagramのデータを第三者へ販売しません。

第三者サービス

本アプリは、YouTube Data API ServicesおよびInstagram APIを使用します。 これらのサービスの利用には、各サービス提供元の規約とプライバシーポリシーが適用されます。

YouTube API Services

本アプリはYouTube Data API Servicesを使用します。ユーザーはGoogleのセキュリティ設定ページから、 いつでも本アプリのGoogleアカウントへのアクセスを取り消すことができます。 Googleのプライバシーポリシーは https://policies.google.com/privacy で確認できます。

データ削除

アプリ内のログアウトまたはSNS削除操作により、端末内に保存された該当SNSのログイン情報と関連データを削除できます。 詳細はデータ削除案内をご確認ください。
